D3 Selections

Try using selections on the h3 elements below. The elements are wrapped in a div whose id is 'section1'. For a full reference on selections, see here

  1. Select the first h3 element using d3.select()
  2. Select all of the h3 elements using d3.selectAll()
  3. Add or remove the 'blue' css class to the h3 elements using selection.classed()
  4. Add an attribute using selection.attr()
  5. Change the text of a header element using selection.text()
  6. Add a new header using selection.append()
  7. Remove a header using selection.remove()

D3 and SVG

D3's real strength is manipulating SVG. Try the items below.

  1. Add an svg element to the div on the right. The div's id is "section2". Save the d3 selection to a variable, it will make it easier later on.
  2. Add a g element to the svg. g elements are like divs for svg's. Save the d3 selection to a variable as well.
  3. Add an svg shape of your choice to the g element

Binding Data

  1. Add an svg element to the div on the right with id = "section3"
  2. Add a g element to the svg element
  3. This site includes data in a file named data.js Bind the dinosaurs array from data.js to some circle elements. Make sure you put the circles inside the g element you created in the previous step
  4. Let's work with data joins. Bind the circles from the previous step to the unicorns array and save the return value to a variable.
    1. The length of the unicorns array is less than the length of the dinosaurs array. Use the exit selection to remove the extra elements
    2. Try using the enter selection to update the appearance of the circles. What do you think will change?
    3. Change the remaining elements with the update selection
  5. Now try adding a transition. Re-bind the circles to the dinosaurs array and transition their radius. You'll probably want to write a script for this. See here for a complete reference on transitions
