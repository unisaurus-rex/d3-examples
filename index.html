Some things to try:
  1. Select the first h3 element using d3.select()
  2. Select all of the h3 elements using d3.selectAll()
  3. Add or remove the 'blue' css class to the h3 elemdents using selection.classed()
  4. Add an attribute using selection.attr()
  5. Change the text of a header element using selection.text()
  6. Add a new header using selection.append()
  7. Remove a header using selection.remove()
